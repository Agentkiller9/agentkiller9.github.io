$ About

I'm a final-year BSc (Hons) Computer Science in Cybersecurity student at Asia Pacific University (APU) / De Montfort University (DMU), currently serving as Head of the SecOps Functional Unit where I lead internal penetration tests on web applications and Active Directory, mentor security interns, and oversee the full assessment lifecycle.

My focus is split between offensive security (web and AD pentesting, red team engagements) and DFIR (threat hunting with Hayabusa/Takajo/Chainsaw, Windows event log forensics). My final-year project, Jinsoku, is a production-ready self-hosted DFIR workbench wrapping Hayabusa and Takajo behind a React + FastAPI front end.

Outside university I run training and CTF infrastructure for CyberNexus as a Board Director, and I sharpen skills on Hack The Box and TryHackMe.
